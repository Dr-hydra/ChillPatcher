### 二、共享内存 PCM 通道

**文件: `backend/Audio/SharedMemoryServer.cs`**

#### 2.1 内存布局

```
         ┌────────────────────────────────────────────┐
 0x0000  │ Magic: "CHILLPCM\0" (8 bytes)              │
 0x0008  │ Version: u32        = 1                    │
 0x000C  │ SampleRate: u32     = 44100               │
 0x0010  │ Channels: u16       = 2                    │
 0x0012  │ BytesPerFrame: u16  = 8 (ch×sizeof(float)) │
 0x0014  │ BufferFrames: u32   = 220500 (5秒)         │
 0x0018  │ PlayState: i32      (0=stop 1=play 2=pause)│
 0x001C  │ SeekFrame: i64      (游戏请求, 后端消费清0) │
 0x0024  │ Flags: u32          (bit0=flush_req,...)   │
 0x0028  │ CurrentUUID: byte[64] (ASCII, 当前曲目)    │
 0x0068  │ WriteCursor: i64    (后端原子写)            │
 0x0070  │ ReadCursor: i64     (游戏原子读)            │
 0x0078  │ GapSeconds: f32     (缓存秒数, 用于预读)    │
 0x007C  │ Reserved[36]                               │
 0x00A0  │ ──────────────────────────────────────      │
         │ PCM Ring Buffer                            │
         │ float[BufferFrames × Channels]             │
         │ = 220500 × 2 × 4 = 1,764,000 bytes         │
         └────────────────────────────────────────────┘
```

#### 2.2 环形缓冲区算法

```csharp
// 后端 (写者)
void WriteFrames(float[] pcm, int frameCount)
{
    var start = WriteCursor % BufferFrames;
    var len = frameCount * Channels;

    if (start + len <= BufferFrames * Channels)
    {
        // 不绕环, 直接拷贝
        Marshal.Copy(pcm, 0, RingBufferPtr + start * 4, len);
    }
    else
    {
        // 绕环, 分两段拷贝
        var first = (BufferFrames * Channels) - start;
        Marshal.Copy(pcm, 0, RingBufferPtr + start * 4, first);
        Marshal.Copy(pcm, first, RingBufferPtr, len - first);
    }

    Interlocked.Exchange(ref WriteCursor, WriteCursor + frameCount);
}

// 游戏 (读者)
int ReadFrames(float[] buffer, int framesToRead)
{
    var avail = Volatile.Read(ref WriteCursor) - Volatile.Read(ref ReadCursor);
    var read = Math.Min(framesToRead, (int)avail);
    // 环形读取逻辑...
    Interlocked.Exchange(ref ReadCursor, ReadCursor + read);
    return read;
}
```

**免锁原理**: 单写者多读者场景下，写者仅更新 `WriteCursor` (原子操作)，读者仅更新 `ReadCursor`。不存在同一位置同时读写，因为 `ReadCursor` 始终追赶 `WriteCursor`，但永远不会超过。读者落后写者至少 `BufferFrames - MaxReadBatch`，天然安全。

#### 2.3 UWP 兼容 ACL

```csharp
// 创建共享内存时设置 ACL
static IntPtr CreateWithAppContainerAcl(string name, int size)
{
    var sid = new SecurityIdentifier("S-1-15-2-1"); // ALL APPLICATION PACKAGES
    var rawSid = new byte[sid.BinaryLength];
    sid.GetBinaryForm(rawSid, 0);

    // EXPLICIT_ACCESS: ALL_APP_PACKAGES → GENERIC_READ + SYNCHRONIZE
    var ea = new EXPLICIT_ACCESS
    {
        grfAccessPermissions = GENERIC_READ | SYNCHRONIZE,
        grfAccessMode = GRANT_ACCESS,
        grfInheritance = NO_INHERITANCE,
        Trustee = new TRUSTEE
        {
            TrusteeForm = TRUSTEE_IS_SID,
            TrusteeType = TRUSTEE_IS_WELL_KNOWN_GROUP,
            ptstrName = rawSid
        }
    };

    SetEntriesInAcl(1, ref ea, IntPtr.Zero, out IntPtr pAcl);
    InitializeSecurityDescriptor(out SECURITY_DESCRIPTOR sd, 1);
    SetSecurityDescriptorDacl(ref sd, true, pAcl, false);

    SECURITY_ATTRIBUTES sa;
    sa.nLength = Marshal.SizeOf<SECURITY_ATTRIBUTES>();
    sa.lpSecurityDescriptor = &sd;
    sa.bInheritHandle = false;

    // Global\ 命名空间 + ACL = 所有进程包括 UWP 可读
    return CreateFileMapping(
        INVALID_HANDLE_VALUE, ref sa,
        PAGE_READWRITE,
        (size >> 32) & 0xFFFFFFFF, size & 0xFFFFFFFF,
        @"Global\OmniMixPlayer_PCM");
}
```

### 八、Slint 优势总结

| 需求              | Slint 方案                                   |
| ----------------- | -------------------------------------------- |
| 一套代码桌面+Web  | 同 `.slint` + 同 Rust 逻辑, 条件编译目标平台 |
| 桌面原生渲染      | Skia 后端, 原生窗口, 系统托盘                |
| WebGL/WebGPU 后端 | `slint::backend::Web` → WASM                 |
| 手机浏览器控制    | WASM 从 HTTP 服务加载, 连局域网              |
| 高性能            | Rust 编译, 无 GC, 渲染开销极低               |
| 开发体验          | VS Code 扩展, 实时预览, 热重载               |
